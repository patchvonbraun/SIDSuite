SIDSuite by patchvonbraun

Tools for SID monitoring using Gnu Radio and Audio input

This is a collection of code, based on Gnu Radio and GRC to allow you to perform SID (Sudden Ionospheric Disturbance) monitoring using an ALSA sound card for input. It includes data-processing tools, and tools for uploading your data to the main SID repository at Stanford University.

Dependencies

Gnu Radio

Install

make -f Makefile.sidsuite make -f Makefile.sidsuite install
